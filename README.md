# Udemy Clone

A simple one-page Udemy clone website built using HTML and CSS. This project recreates the basic layout of the Udemy homepage with course sections, categories, and a clean user interface. It helped me improve my frontend development and styling skills.

## Technologies Used
- HTML5
- CSS3

## Features
- Responsive one-page design
- Navigation bar
- Course cards section
- Categories section
- Simple and clean layout

## How to View
1. Download or clone this repository
2. Open `index.html` in your browser
